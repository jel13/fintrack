# **App Name**: FinTrack Mobile

## Core Features:

- Expense Tracking: Allow users to input and categorize their income and expenses.
- Budgeting: Set monthly budget limits for different expense categories and display a progress bar.
- Spending Analysis: Visualize spending patterns through charts and graphs, showing where money is going.

## Style Guidelines:

- Primary color: Calm blue (#3498db) for trustworthiness and stability.
- Secondary colors: Light gray (#ecf0f1) for backgrounds and card separation.
- Accent: Green (#2ecc71) for positive indicators (e.g., savings, budget surplus).
- Clean and readable typography for data clarity.
- Simple and intuitive icons for categories and actions.
- Clean and organized layout for easy navigation.
